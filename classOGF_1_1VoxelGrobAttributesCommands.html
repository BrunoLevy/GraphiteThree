Reconstructs a surface from points and normals using Misha Kahzdan's Screened Poisson Reconstruction.

Parameters
[in]pointsthe name of the object that stores the points. It needs to have a "normal" attribute.
[in]attributethe name of the attribute used to store the reconstructed implicit function.
[in]depththe depth of the octree, 8 is the default value, use 10 or 11 for highly detailed models
[in]reconstruction(optional) the name of the reconstructed surface
